# Reference Paper Manifest

Date: 2026-06-22

This directory is for local copies of papers used while designing social-media
simulation experiments. PDF files are intentionally git-ignored because the
current local set is about 69 MB. Track source URLs and reading roles here
instead of committing binaries.
